# Currency Converter (API-Based Web App)

A web-based currency converter built using **HTML, CSS, and JavaScript** that allows users to convert between multiple currencies in real-time. The application fetches live exchange rates using a currency API, ensuring accurate and up-to-date conversions.

---

## Features
- Convert between **any two currencies** with up-to-date exchange rates.
- Real-time API integration to fetch accurate currency data.
- User-friendly input fields and dropdowns for selecting currencies.
- Handles decimal values and invalid inputs.
- Responsive design for desktop and mobile screens.

---

## Technologies Used
- **HTML & CSS**: For layout, styling, and responsive design.
- **JavaScript**: Handles API requests, user input, and conversion logic.
- **API Integration**: Fetches live currency exchange rates.
